# Pokédex - Pokémon Game and Movie Companion

Este é um projeto da Pokédex em Kotlin que utiliza a API do [PokeAPI](https://pokeapi.co/) para exibir informações sobre Pokémon do jogo e filmes. O aplicativo apresenta uma interface de usuário amigável com cards de Pokémon em um RecyclerView, e as informações são buscadas e exibidas utilizando a biblioteca Retrofit.

## Funcionalidades

- Visualize informações detalhadas sobre diferentes Pokémon, incluindo nome, tipo e muito mais.
- Explore uma lista de Pokémon em cards interativos.
- Obtenha dados precisos e atualizados do PokeAPI.

## Tecnologias Utilizadas

- **Kotlin**: A linguagem de programação principal do aplicativo.
- **SDK Version 33**: O projeto foi compilado com a versão 33 do Android SDK.
- **Retrofit**: Uma biblioteca amplamente utilizada para fazer chamadas de API HTTP em aplicativos Android.
- **Glide**: Uma biblioteca de carregamento de imagens para exibir as imagens dos Pokémon.

## Como Executar

Para executar este projeto em sua máquina local, siga estas etapas:

1. Clone este repositório:

   ```git clone https://github.com/CaianMarcinkowski/pokedex-kotlin.git```

    Abra o projeto em seu ambiente de desenvolvimento Kotlin (como o Android Studio).

    Execute o aplicativo em um emulador Android ou dispositivo físico.

    Comece a explorar a Pokédex e aprender mais sobre os incríveis Pokémon do mundo dos jogos e filmes!
